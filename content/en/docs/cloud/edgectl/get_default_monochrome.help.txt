Get edgectl default monochrome. Controls whether output is monochrome (or color).

Usage:
  edgectl get default monochrome [flags]

Examples:
  $ edgectl get default monochrome

Flags:
  -h, --help   help for monochrome

Global Flags:
  -H, --header         Print header row
  -j, --json           JSON output
  -M, --monochrome     Don't colorize output
  -t, --text           Text table output
      --token string   Access Token
  -v, --verbose        Verbose output
