Set edgectl default header.  Controls whether header row is shown.

Usage:
  edgectl set default header true|false [flags]

Examples:
  $ edgectl set default header true

  $ edgectl set default header false

Flags:
  -h, --help   help for header

Global Flags:
  -H, --header         Print header row
  -j, --json           JSON output
  -M, --monochrome     Don't colorize output
  -t, --text           Text table output
      --token string   Access Token
  -v, --verbose        Verbose output
